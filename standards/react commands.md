# Most Used Git CommandsReact Commands

## 1. Create React Project + Vite

```sh
# Create a new React project using Vite
npm create vite@latest my-app
```

```sh
# Navigate to project, install dependencies and start dev server
cd my-app
npm install
npm run dev
```

## 2. Development Commands

```sh
# Start development server only on localhost
npm run dev
```

```sh
# Start development server and expose it to the local network
npm run dev -- --host
```

```sh
# Install project dependencies
npm install
```

```sh
# Update dependencies
npm update
```

## 3. Build and Preview

```sh
# Create production build
npm run build
```

```sh
# Preview production build locally
npm run preview
```
